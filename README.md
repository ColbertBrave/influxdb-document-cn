# InfluxDB中文文档

InfluxDB是一个用于存储和分析时间序列数据的开源数据库。

主要特性有：  

* 内置HTTP接口，使用方便
* 数据可以打标记，这样查询可以很灵活
* 类SQL的查询语句
* 安装管理很简单，并且读写数据很高效
* 能够实时查询，数据在写入时被索引后就能够被立即查出
* ……

在最新的[DB-ENGINES](https://db-engines.com/en/ranking/time+series+dbms)给出的时间序列数据库的排名中，InfluxDB高居第一位，可以预见，InfluxDB会越来越得到广泛的使用。
